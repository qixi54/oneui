MermaidChart 图表组件
使用说明

MermaidChart 组件通过引入 Mermaid CDN 渲染流程图、时序图、甘特图等。在 Vue 组件中使用如下：

Vue 组件引入示例 
<!-- index.html 引入 Mermaid CDN -->
<script src="https://cdn.jsdelivr.net/npm/mermaid@10/dist/mermaid.min.js"></script>

<!-- Vue 组件使用 -->
<MermaidChart :code="flowCode" />

<!-- flowCode 示例值 -->
const flowCode = `
graph TD
  A[开始] --> B{条件判断}
  B -->|是| C[执行操作]
  B -->|否| D[跳过]
  C --> E[结束]
  D --> E
`
支持的图表类型
Flowchart · 流程图
graph TD / LR / RL
Sequence · 时序图
sequenceDiagram
Gantt · 甘特图
gantt
Class / Entity · 类图 / ER图
classDiagram / erDiagram
实际渲染需在页面引入 Mermaid CDN（约 500KB），建议按需异步加载。
